# 🚌 Sahi Bus Platform

A modern web platform designed to **compare, explore, and analyze bus transport options** efficiently.  
Built using **Next.js**, it provides a fast, responsive, and intuitive user experience for real-world travel data.

---

## 🌐 Live Demo

🔗 **Public Website:** [Sahi Bus Platform](https://damp-graveyard-564pq7j467pcjpx-9002.app.github.dev/)

---

## 🚀 Features

- ⚡ **Next.js 14 Framework** — for ultra-fast rendering and routing  
- 🎨 **Modern UI/UX** — clean, responsive interface  
- 🔍 **Bus Comparison Engine** — easily compare routes, prices, and operators  
- 🧭 **Smart Filters & Sorting** — quickly find the best travel options  
- 📊 **Analytics & Insights** — visualize route data and performance trends  
- 🤖 **AI-Ready Architecture** — prepared for future intelligent features

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/adimavic/Sahi-bus-plaform.git
cd Sahi-bus-plaform
````

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start the Development Server

```bash
npm run dev
```

Visit: [http://localhost:3000](http://localhost:3000)

### 4️⃣ Build for Production

```bash
npm run build
npm start
```

---

## 🧩 Requirements

* **Node.js:** `>= 18.18.0`
* **npm:** `>= 8.19.4`

> ⚠️ If you see an error like
> “Next.js requires Node.js version ^18.18.0 || ^19.8.0 || >= 20.0.0”
> — please upgrade your Node version.
